# go语言八小时入门课程

## 初衷

之前一版的go基础课程，课程总时长达到了12小时

对我来说，还是有点太长了

并且有些该深入的地方没有深入



那么本课程的初衷是要把时间压缩到8小时左右

减少不必要的学习成本，真正做到该讲的讲，不该讲的不讲

并且在讲解原理的地方，尽可能结合图形的方式

让大家能够真正理解
